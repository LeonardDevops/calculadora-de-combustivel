Calculadora de Combustível
Uma aplicação simples desenvolvida em ReactJS com TypeScript para ajudar os usuários a determinar qual tipo de combustível é mais vantajoso financeiramente: álcool ou gasolina.

Descrição
A Calculadora de Combustível é uma aplicação web desenvolvida utilizando a biblioteca ReactJS e TypeScript para criar uma interface amigável e responsiva. Esta aplicação permite aos usuários inserir o preço do álcool e da gasolina e calcula qual deles é mais vantajoso considerando o preço por litro de cada combustível.

Funcionalidades
Cálculo de Melhor Combustível: A aplicação realiza um cálculo simples para determinar se o álcool ou a gasolina é mais vantajoso, com base nos preços inseridos pelo usuário.
Tipagem com TypeScript: Todo o código é escrito utilizando TypeScript, garantindo uma melhor organização e prevenindo erros comuns.
Gerenciamento de Estado com useState: Utilização do hook useState do React para gerenciar o estado da aplicação de forma eficiente.
Tipagem de Estados e Props: Tipagem das variáveis de estado e props com interfaces para garantir uma codificação mais segura.
Estilização com CSS: Aplicação de estilos utilizando CSS para uma interface visualmente agradável e responsiva.
